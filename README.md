### 配置文件说明
项目部署至服务器后，需修改各类配置文件替换域名、资源链接，各文件作用如下：
1. **云喵A.txt**：版本检测配置，填写版本号、更新内容、下载链接、备用链接
2. **云喵B.txt**：页面随机文案、工具内容，一行一条，可直接使用官方线上数据源
3. **官网链接**：全局多页面共用，替换为自己的项目域名
4. **云控.js**：自定义脚本，可实现版本检测、广告投放、黑名单管控等功能
5. **公告.txt**：首页公告，自动识别换行展示
6. **轮播图.txt**：首页轮播素材，图片链接+跳转链接为一组，可无限新增
7. **捐赠名单.txt**：展示捐赠记录，格式：用户名：/QQ号：/金额：，自动读取QQ头像
8. **特殊用户.txt**：自定义用户专属称号，主页、排行榜展示，格式：QQ号：/展示信息：
9. **猫粮账户.txt（核心配置）**：用户资产数据，格式：账号：/猫粮余额：0/账户状态：正常
10. **云喵C.txt**：免验证码登录授权码，格式：授权码：/QQ号：，建议使用纯数字授权码
11. **工具.txt**：工具库配置，先填写分类名称，下方填写工具名与跳转链接；分类需配套同名png图标，可调用官方工具列表
12. **壁纸.txt**：壁纸资源配置，格式：分享人：/图片链接：，支持官方线上壁纸数据源
13. **商店.txt**：商城商品配置，格式：商品名称：/价格：/剩余库存：
14. **支持名单.txt**：项目支持者展示列表，格式：名称：/QQ号：/类型：
15. **签到数据.txt**：用户签到记录，控制每日抽奖权限，单条一行填写签到时间与QQ账号
16. **每日抽奖.txt**：抽奖奖品概率设置，格式：奖品名称/概率
17. **活动.txt**：运营活动配置，格式：活动主题：/活动内容：
18. **授权.txt**：第三方应用QQ授权管理，状态1=正常，2=封禁
19. **贡献值.txt**：贡献排行榜数据，格式：QQ号：/贡献值：
20. **口令访问.txt**：隐藏页面口令入口，格式：口令：/目标页面：
21. **域名**：填写项目主域名（示例：ymkj.bid）
